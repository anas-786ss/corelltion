# Supply Chain Analytics: Correlation Analysis

**Author Email:** 23f3003127@ds.study.iitm.ac.in

## Project Overview
This repository contains the correlation analysis of key supply chain metrics for a major automotive manufacturer. The analysis identifies relationships between supplier lead times, inventory levels, order frequency, delivery performance, and unit costs.

## Files Description
1.  **correlation.csv**: The calculated correlation matrix containing Pearson correlation coefficients for all variable pairs.
2.  **heatmap.png**: A visualization of the correlation matrix using a Red-White-Green color scale (Red = Low Correlation, Green = High Correlation).

## Analysis Summary
The analysis was performed using standard statistical methods to support data-driven decision-making in procurement and inventory management.
